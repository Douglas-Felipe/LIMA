# Trabalho de LIMA - Jogo de quiz
Neste trabalho eu utilizei cards em cada quiz, no seu HTML.
![Imagem da tela inicial do Quiz](./img/telaInicialQuiz.png)
No CSS utilizei tags para fazer um "Flip" das cards.
